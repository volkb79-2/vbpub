## Decisions

- Decision: Use module-level ActionKind enum in catalog.py rather than a free-form string registry.
  Reason: Allowlist enforcement — unknown kinds are rejected at import time.
  Impact: Simple, typed, easy to extend later without changing CLI dispatch.

- Decision: Audit records are written only when --audit-log PATH is given.
  Reason: The handoff spec says "if --audit-log PATH is provided" — no implicit files.
  Impact: Clean separation; no side effects without explicit opt-in.

- Decision: Command previews are lists of strings (argv), never shell strings.
  Reason: Contract established in P21 handoff: "Build command previews as argv lists, never shell strings."
  Impact: Subprocess invocation is obviously absent.
